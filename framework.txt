Nama framework : Angular 1

Definisi :
AngularJS adalah framework open source berbasis javascript terstruktur dengan konsep MVC untuk front-end pada web app yang bersifat dinamis dimana menggunakan sintaks HTML dengan jelas dan ringkas. Dua arah data binding dan dependencies pada AngularJS dapat memperingkas penulisan code dan seluruh proses terjadi pada browser sehingga dapat menjadi framework yang dapat dipadukan pada beragam teknologi di server side.

Sejarah :
AngularJS 1 pertama kali dikemabangkan oleh Misko Hevery & Adam Abrons pada tahun 2009.

Alasan framework ini dibuat :
AngularJS merupakan project dari Google namun sekarang sudah merupakan project open source, tujuannya untuk merubah static HTML menjadi dynamic HTML yang memiliki kemampuan untuk dapat menambah kemampuan attributes dan component dari HTML dan menyediakan custom attributes menggunakan javascript dan dapat dengan cepat digunakan untuk tahap development.
